## ORMS (Beta version)

> ORMS Dashboard made with Material-UI components and React.

## Getting started

- Recommended `node js 14+` and `npm 6+`
- Install dependencies: `npm install` or `yarn install`
- Start the server: `npm run start` or `yarn start`
